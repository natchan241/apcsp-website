Common Linux Commands

This is a list of 10 of the most common Linux commands. They can be used on the Raspberry Pi and/or the Macbook's Terminal.

Name of Command Brief Description Example of Common Usage
cd Go to home directory To go back to home: "cd"
ls -al Show a list of the home directory (while in the home directory) To show what items are within the home directory (files, directories, etc.)
pwd See which directory you are under To make sure you are working under the right directory when saving files and accessing them
nano A file editor To edit files and save them to the directory (nano rpistatus edits that file)
cd/(filename) Goes to that directory/folder To quickly access that directory (cd/scripts goes to that directory)
man (command) Manual: shows how to use that command (man ping) tells you how the ping command is used and what parameters are associated with them
uname Print the operating system name Shows the operating system name as requested (ex. Darwin)
echo (string) Prints the inputted string (echo Hello) will print Hello
cp (file/directory) (destination) Copies file or directory to the destination (cp rpistatus Scripts) copies the file name rpistatus to Scripts
rm (file) OR remdir Removes the inputted file OR Removes the inputted directory (rm rpistatus) removes the file name rpistatus
